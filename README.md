# scala-cli-sbt-crypto-scrypt-hashing

## Description
A demo for Scrypt one-way hashing of
a password.

## Tech stack
- sbt
- scala

## Docker stack
- openjdk:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
